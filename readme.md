# GitHub User Finder

A responsive GitHub User Finder web app that fetches and displays real-time GitHub profile data and top repositories using the GitHub API.

## 🚀 Features

- Search any GitHub username
- Fetches real GitHub profile data using API
- Displays:
  - Profile image
  - Name
  - Username
  - Location
  - Website
  - Join date
  - Public repositories
  - Followers
  - Following
- Shows top 4 repositories based on stars
- Handles invalid username errors
- Responsive design for desktop and mobile

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript
- GitHub REST API
